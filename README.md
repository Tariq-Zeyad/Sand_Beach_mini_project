# 🏖️ Sand Beach Restaurant Website


## 🌟 Overview

**Sand Beach Restaurant** is a fully responsive restaurant website designed for a seafood restaurant located in **Gaza, Khan Younis - South of Al-Nus Intersection, By the Sea**. The website showcases the restaurant's menu, location, and provides an online reservation system.

## ✨ Features

### Core Features
- ✅ **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- ✅ **Bootstrap 5 Components** - Navbar, Cards, Accordion, Forms, Carousel
- ✅ **Modular Architecture** - Each section is a separate component file
- ✅ **Dynamic Component Loading** - JavaScript loads components dynamically
- ✅ **Pure CSS Alternative** - Zero JavaScript version available
- ✅ **Online Reservation** - Table booking form with validation
- ✅ **Menu Filtering** - Filter dishes by category (Seafood, Grill, Salad, Dessert)
- ✅ **Customer Testimonials** - Reviews carousel with ratings
- ✅ **FAQ Section** - Accordion with common questions
- ✅ **Smooth Scrolling** - Navigation links scroll smoothly to sections
- ✅ **Active Navigation** - Current section highlighted in navbar
- ✅ **Print Styles** - Optimized for printing menus

### Design Features
- 🎨 **Ocean-Themed Colors** - Blue, sand, and coral color palette
- 🎨 **CSS Variables** - Easy color customization
- 🎨 **Gradient Backgrounds** - Modern visual effects
- 🎨 **Hover Animations** - Interactive card and button effects
- 🎨 **Box Shadows** - Depth and elevation effects
- 🎨 **Rounded Corners** - Soft, modern design language
- 🎨 **Mobile-First** - Optimized for all screen sizes
